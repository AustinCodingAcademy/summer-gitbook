Day 3

Objectives

  • Students will build a website for their business using HTML and CSS.
  • Students will deploy their website using Git and Github.

Agenda

  • Website Build
  • Deployment

Website Build

  1. Login to your account on github.
  2. Create a new repo.
  3. Follow the steps to clone the repo to your computer.
  4. Inside of your project folder, add a file called index.html.
  5. Inside of your project folder, add a file called style.css.
  6. Follow along with instructor to build your website.

Deployment

  1. Follow along with your instructor to deploy your website using github pages.
